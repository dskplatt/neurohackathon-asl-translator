# data/

This directory contains training and calibration data files.
These are not committed to the repository.

## Files

| File | Generated by | Description |
|---|---|---|
| calibration_data.csv | scripts/collect_calibration_data.py | Personal EMG recordings |
| combined_dataset.csv | training/combine_dataset.py | Combined multi-user dataset |
| windows.npz | training/preprocess.py | Pre-windowed training data |

## Collecting calibration data

```bash
python scripts/collect_calibration_data.py
```

This guides you through signing each ASL letter and saves your
personal EMG recordings to calibration_data.csv.
